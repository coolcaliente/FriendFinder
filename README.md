# FriendFinder
Answer some simple questions to be matched with the perfect friend.  

Find your a friend fit here:
https://mighty-meadow-48808.herokuapp.com/

## Instructions

Fill out all the questions on the survey and press submit to meet your new bestie. 

## Technologies Used

- **JavaScript** - Makes the website interactive.
- **Bootstrap** - Used to create the grid and add css styles.
- **Node** - Uses Javascript on the server side.
- **Express** - Fast, unopinionated, minimalist web framework for node.
- **NPM Packages** - express and body-parser.
